What is a Queue?

A queue is similar to a stack in a way that it is also a 2d list-like structure. The only diffference is that queue's push and pop operation is FIFO(First in First Out) so that items go out from a different side comparing to a stack.

How did I implement a queue in Javascript/Typescript?

It is tempting to just use a JavaScript Array, the same structure we used to build a stack, to build a queue. We could use Array.shift() which directly removes the item from the front of the array. However, this operation takes O(n) Time complexity, while we want O(1) for both push and pop. Therefore the Javascript built in Array is not an ideal data structure to implement a queue.
